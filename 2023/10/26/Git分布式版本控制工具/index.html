Git分布式版本控制工具

1、目标

  • 了解git基本概念
  • 能够概述git工作流程
  • 能够使用git常用命令
  • 熟悉git代码托管服务
  • 能够使用idea操作git

2、概述

==git remote add origin 链接== 绑定远程仓库

==git push –set-upstream origin master== 将master分支推送到远程仓库并绑定

==git clone 地址== 将远程仓库克隆到本地文件夹下

==git push origin master== 将分支推送到远程仓库

==git pull== 同步远端代码
